# 🏢 AlquilerApp - Gestión de Inmuebles (Admin Workspace)

Plataforma unipersonal para la gestión integral de departamentos, inquilinos, cobros de servicios y generación de comprobantes.

## 🚀 Stack Tecnológico
* **Frontend:** React (Vite) + Tailwind CSS
* **Backend & DB:** Firebase (Auth, Firestore, Storage, Cloud Functions)
* **Generación de PDF:** `@react-pdf/renderer` o `jspdf`
* **Mensajería:** Integración nativa con `api.whatsapp.com`

## 📁 Estructura del Proyecto
├── /src
│   ├── /assets        # Iconos, logos y plantillas estáticas
│   ├── /components    # Componentes UI reutilizables (Botones, Tarjetas, Modales)
│   ├── /config        # Configuración de Firebase y variables de entorno
│   ├── /context       # Estado global (AuthContext, AppContext)
│   ├── /hooks         # Custom hooks para la lógica de Firestore
│   ├── /pages         # Vistas principales (Dashboard, Inmuebles, Configuración)
│   ├── /services      # Lógica de Cloud Functions y utilidades de PDF
│   ├── /utils         # Formateadores de moneda, fechas y validaciones
│   ├── App.jsx        # Enrutador principal
│   └── main.jsx       # Entry point
