# Árbol de Componentes React

La aplicación está dividida en contenedores lógicos para maximizar la reutilización.

## Layout Principal
* `AppLayout.jsx` (Contiene el Sidebar de navegación y el Header)
  * `Sidebar.jsx` (Navegación: Dashboard, Departamentos, Inquilinos, Configuración)

## Vistas (Pages)
* `Dashboard.jsx`
  * `MetricCards.jsx` (Tarjetas de totales: Ocupados, Ingresos, Pendientes)
  * `QuickActionsList.jsx` (Lista de departamentos con botones de cobrar)
* `Departamentos.jsx`
  * `DeptoGrid.jsx` (Lista visual de los inmuebles)
  * `DeptoFormModal.jsx` (Formulario para crear/editar con upload de fotos)
* `InquilinoDetail.jsx` (La vista de "Ficha Gradual")
  * `DataBasicaForm.jsx`
  * `DocumentUploadBox.jsx` (Drag & drop para DNI y Contrato)
  * `ServiciosCalculadora.jsx` (La tabla de lecturas de luz y agua)
* `Configuracion.jsx`
  * `CuentasBancariasForm.jsx`

## Utilidades y Servicios
* `PDFGenerator.jsx` (Plantilla React-PDF que recibe los datos para renderizar la boleta A5)
* `WhatsAppLinkBuilder.js` (Función pura que concatena los datos financieros, formatea el texto con saltos de línea y retorna la URL `https://wa.me/...`)
