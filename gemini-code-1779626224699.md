# 🚀 Instrucciones de Desarrollo para Agente

1. **Aislamiento (Security First):** Antes de cualquier operación, verifica que exista un `session.user.id`. Todas las tablas en Supabase tienen habilitado RLS (Row Level Security).
2. **Navegación:** La App debe ser tipo "Single Page" para evitar confusiones. Un menú inferior con 3 botones gigantes: [Inicio] [Departamentos] [Mi Cuenta].
3. **Manejo de Errores:** Si Supabase falla, muestra un mensaje gigante: "⚠️ Hubo un problema al guardar. Revisa tu conexión a internet".
