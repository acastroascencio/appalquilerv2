# Esquema de Base de Datos (Firestore)

El sistema utiliza validaciones estrictas y carga gradual de documentos.

## Colección: `Propiedades`
* `id`: Document ID
* `identificador`: String (Ej. "2A", "3B")
* `estado`: Boolean (true: Ocupado, false: Disponible)
* `costo_base`: Number
* `caracteristicas`: Object
  * `habitaciones`: Number
  * `bano_propio`: Boolean
  * `cocina`: Boolean
* `fotos`: Array de Strings (URLs de Firebase Storage)

## Colección: `Inquilinos`
* `id`: Document ID
* `propiedad_id`: Referencia a `Propiedades`
* `nombre`: String
* `telefono`: String
* `garantia_monto`: Number
* `documentos`: Object (Carga Gradual)
  * `dni_url`: String (Opcional)
  * `contrato_url`: String (Opcional)
* `vehiculo`: Object (Informativo - Excluido de balances)
  * `tiene_vehiculo`: Boolean
  * `tipo`: String
  * `placa`: String
  * `monto_asociacion`: Number

## Colección: `Mensualidades` (Servicios y Cobros)
* `id`: Document ID
* `inquilino_id`: Referencia a `Inquilinos`
* `mes_anio`: String (Ej. "05-2026")
* `estado`: String ("Pendiente", "Pagado")
* `servicios`: Object
  * `luz`: Object { aplica: Boolean, lectura_anterior: Number, lectura_actual: Number, subtotal: Number }
  * `agua`: Object { aplica: Boolean, lectura_anterior: Number, lectura_actual: Number, subtotal: Number }
  * `seguridad`: Object { aplica: Boolean, subtotal: Number }
* `total_cobrado`: Number (Calculado automáticamente)
* `voucher_url`: String (Opcional)

## Colección: `Configuracion` (Documento Único: `admin_settings`)
* `titular`: String ("Mario Andres Castro Ascencio")
* `billeteras`: Object { yape: String, plin: String }
* `bancos`: Object { bcp_cuenta: String, bcp_cci: String, interbank_cuenta: String, interbank_cci: String }
